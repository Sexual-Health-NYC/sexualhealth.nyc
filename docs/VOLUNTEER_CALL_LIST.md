# Volunteer Call List

75 clinics total. Pick one, call, verify everything. See [VOLUNTEER_GUIDE.md](VOLUNTEER_GUIDE.md) for the script.

**Why verify even if we have data?** Websites and directories go stale. Hours change seasonally. Phones get reassigned. The only way to know for sure is to call.

Clinics marked with ⚠️ have **no hours on file** — these are highest priority.
